
# Hotel Management Website

This is a static website like Galaxy Hotel Vadodara. On this Website, the Most four Option is their home, services, rooms, and customers. Services like food services, refreshment, housekeeping, and room security. another functionality is the room booking. some photo of the room is available on this website.
